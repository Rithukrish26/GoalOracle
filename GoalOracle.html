GoalOracle

What is GoalOracle?

GoalOracle is an app made using the coding language Python that predicts football scores using just a few stats and mathematical formulas.

Mathematical Formulas

  • Poisson Distribution
  • Matrix Operations
  • Probability Theory
  • Basic Arithmetic Formulas

Stats Needed

  • Goals Scored
  • Goals Conceded
  • Shots on Target
  • Chances Created
  • Possession
  • Pass Completion
